Bagaimana Logika "Automatic Off" Kegiatan Berjalan?

  • Di database, setiap kegiatan memiliki field startDateTime dan endDateTime.
  • Di query Next.js Server Components untuk Landing Page, kita akan menambahkan filter database seperti: 
    where: {
  endDateTime: {
    gte: new Date() // Hanya tampilkan kegiatan yang belum berakhir
  }
}
  • Dengan cara ini, kegiatan yang sudah lewat waktunya tidak akan membebani tampilan depan, tanpa perlu admin mematikannya secara manual!
